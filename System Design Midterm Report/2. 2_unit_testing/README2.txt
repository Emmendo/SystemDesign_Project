Unit Testing:
Definition - Small tests that result from a single unit (such as a function)


- Logging in (creds are stored in the SQL Database that is connected at the beginning of the source code): If the creds are correct, you will be able to log in, if not it will say the creds are invalid.

- Classes Enrolled/Managed: Shows a dynamic list of the courses either enrolled (as a student user) or managed (as a teacher user). This corresponds with the database and which classes each user is associated with.

- Past/Upcoming Assignments: Shows a dynamic list of past assignments for both student and teacher, based on the database tables with a grade/feedback provided by the instructor. Upcoming assignments are shown as well and on the teacher side shows when and if the student has submitted yet before the due date.
